# CleanMela – AI-Based Smart Waste Management System

CleanMela is an AI-powered IoT waste management system that provides real-time garbage monitoring, smart alerts, and heatmap analytics for efficient waste collection.

## Live Demo
🔗 [View Project](https://madocks.ai/raj/index.php)

## Features
- Real-time Garbage Level Detection
- Smart Dustbin Monitoring
- AI-based Garbage Prediction
- Voice Alert System
- Heatmap Visualization
- Cloud Data Storage
- Location Tracking

## Technologies Used
- ESP8266
- Ultrasonic Sensor (HC-SR04)
- Firebase
- Python
- Machine Learning
- JavaScript
- Google Maps API

## Working
1. Ultrasonic sensor detects garbage level
2. ESP8266 sends data to cloud
3. AI predicts garbage accumulation
4. Dashboard displays live status & heatmap
5. Alerts generated when bins are full
